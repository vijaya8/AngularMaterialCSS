CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media CSS saves a lot of work. It can control the layout of multiple web pages all at once External stylesheets are stored in CSS files. This is a demonstration of how different stylesheets can change the layout of your HTML page. You can change the layout of this page by selecting different stylesheets in the menu, or by selecting one of the links.

Why Use CSS?

CSS is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes.

HTML

HTML was NEVER intended to contain tags for formatting a web page! HTML was created to describe the content of a web page, using h1 and p tags.

HEAD

The HTML HEAD tag element is a container for metadata (data about data) and is placed between the html tag and the body tag. HTML metadata is data about the HTML document. Metadata is not displayed.

TITLE

The title tag is required in all HTML documents and it defines the title of the document.

The title tag element:

1.defines a title in the browser toolbar.
2.provides a title for the page when it is added to favorites.
3.displays a title for the page in search-engine results.

CSS How To...
